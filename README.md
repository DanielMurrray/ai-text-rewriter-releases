# AI Text Rewriter — Releases

Public release artifacts for [AI Text Rewriter](https://github.com/DanielMurrray/ai-text-rewriter-mac) (macOS menu-bar rewrite tool).

## Contents

- **`appcast.xml`** — Sparkle appcast manifest. Consumed by the installed app to check for updates.
- **GitHub Releases** — `.dmg` installers for each published version, cryptographically signed with Sparkle's EdDSA key.

## Install

Download the latest `.dmg` from the [Releases page](https://github.com/DanielMurrray/ai-text-rewriter-releases/releases/latest), open it, and drag the app to Applications.
